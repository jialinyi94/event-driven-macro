# Event-Driven 宏观经济分析框架

基于高盛、摩根士丹利和摩根大通三大投行的分析方法论，用四层结构对宏观经济事件进行系统性分析。

## 四层分析框架

### 第一层：事件触发与分类

识别事件类型，决定后续传导路径：
- **央行政策**：FOMC/ECB/BoE/BoJ 利率决议、点阵图、QE/QT、前瞻指引
- **经济数据**：就业（NFP/JOLTS/初请）、通胀（CPI/PPI/PCE）、增长（GDP/零售/PMI）
- **地缘政治**：军事冲突、关税制裁、选举。**必须立即用Tier系统判断供给中断等级**
- **市场结构**：国债拍卖、季末再平衡、企业税缴纳日、期货展期
- **结构性趋势**：AI革命、能源转型（长周期）

**地缘事件关键决策点：** 遇到地缘政治事件时，首先读取 `references/tier-system.md` 进行Tier分级（Tier 0/1/2）。Tier判断是整个分析的分水岭——Tier 0/1下"避险买债"成立，Tier 2下通胀逻辑从T+0即反转债券方向。

### 第二层：传导机制

事件激活的渠道（可多个同时）：
- **利率预期重定价** → 收益率曲线移动
- **流动性条件变化**（Fed资产负债表、TGA、RRP）→ 系统流动性总量
- **风险溢价调整** → 风险资产价格
- **通胀与增长预期修正** → 基本面预期
- **资本流动转向** → 汇率与股市
- **石油美元需求冲击**（仅Tier 2）→ USD与油价同向走强

### 第三层：跨资产影响

| 资产类别 | 关注维度 |
|---------|---------|
| **收益率曲线** | 绝对水平；2s10s利差；短端vs长端分化 |
| **信用利差** | IG（融资条件）；HY（风险偏好晴雨表，股市领先指标） |
| **股票** | 估值（利率→P/E）；盈利预期；风格/板块轮动 |
| **外汇** | DXY；G10利差驱动；EM资本流动敏感 |
| **大宗商品** | 原油（供给路径）；工业金属（增长领先）；黄金（实际利率反向） |
| **波动率** | VIX；MOVE（债券波动率溢出至VIX） |

### 第四层：反馈与二阶效应

- **金融条件反馈**：利差扩大/股市下跌 → 金融条件收紧 → 经济放缓
- **仓位调整**：拥挤仓位平仓 → 放大波动（CTA顺势效应）
- **央行反应函数**：金融条件过度收紧 → Fed Put
- **财富效应**：资产价格变化 → 消费支出 → 反哺经济

## 分析输出结构

**① 事件分类与等级** — 事件类型 + 重要性（地缘事件须含Tier判定）
**② 核心传导路径** — 用"A → B → C"因果链，说明速度（即时/数周/数月）
**③ 跨资产影响预测** — 按资产类别列方向+理由
**④ 三大投行视角融合** — GS/MS/JPM各自观点和分歧
**⑤ 关键监控指标** — 2-3个验证/推翻当前分析的指标
**⑥ 体制判断**（地缘/供给冲击必含）— 纯避险/滞胀/恐慌出清？60/40是否有效？做多还是做空债券？
**⑦ 尾部风险提示** — 关键假设和风险情景
