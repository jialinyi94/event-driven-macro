# event-driven-macro\n\nEvent-driven macroeconomic analysis framework skill for Claude.\n\nBased on Goldman Sachs, Morgan Stanley, and JP Morgan methodologies — uses a 4-layer structure:\n1. **Event Triggers** — Classify the macro event (Fed policy, economic data, geopolitics, market structure)\n2. **Transmission Mechanisms** — Identify which channels are activated (rate repricing, liquidity, risk premium, etc.)\n3. **Asset Price Impact** — Predict directional moves across yield curve, credit, equities, FX, commodities, vol\n4. **Feedback Loops** — Second-order effects (financial conditions, forced repositioning, Fed put, wealth effect)\n\nIncludes real historical event validation against actual T+1 and T+5 price data.\n\n## Install\n\n```bash\nskills install jialinyi94/event-driven-macro\n```\n
