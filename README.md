# event-driven-macro

Event-driven macroeconomic analysis framework skill for Claude.

Based on Goldman Sachs, Morgan Stanley, and JP Morgan methodologies — uses a 4-layer structure:
1. **Event Triggers** — Classify the macro event (Fed policy, economic data, geopolitics, market structure)
2. **Transmission Mechanisms** — Identify which channels are activated (rate repricing, liquidity, risk premium, etc.)
3. **Asset Price Impact** — Predict directional moves across yield curve, credit, equities, FX, commodities, vol
4. **Feedback Loops** — Second-order effects (financial conditions, forced repositioning, Fed put, wealth effect)

Includes real historical event validation against actual T+1 and T+5 price data.
